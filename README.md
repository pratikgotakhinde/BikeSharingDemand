Bike Sharing Demand Dashboard

An interactive Streamlit dashboard for analyzing bike rental patterns in Washington, D.C.

Features

- Interactive filters (Year, Season, Weather, Working days, Hour range)
- Time-based analysis with hourly and daily patterns
- Season and weather impact visualization
- Correlation analysis between features

## Data

The dashboard expects a `train.csv` file with bike sharing data including datetime, weather, season, and rental count information.
